Reference Guide - ERPsim for SAP HANA

The purpose of the following guides is to enable you to access data from a SAP HANA database using data sources specifically designed for ERPsim. Refer to the Reference Guide for an overview of the available sources with their description and structure, as well as a few examples of the knowledge you can extract from them. Below you can also find a list of tested tools (including download links) and specific instructions to access data.
